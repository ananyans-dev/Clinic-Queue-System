# Clinic-Queue-System
A Python-based Clinic Queue System developed to simplify patient registration,doctor management, appointment scheduling, and billing using SQLite.
Features
*Patient registration and record management
*Doctor Management
*Appointment Scheduling
*Billing Management
*SQLite Database Integration
Technologies used
*Python
*SQLite
*VS Code
Project Structure
*main.py-Main Application
*doctor.py-Doctor management
*database.py-Database operations
*ui.py-User Interface
*reset_data.py-Reset database
How to Run
1 Clone this respository.
2 Open the project in VS code.
3 Run main.py.
4 Make sure Python is installed on your system.
